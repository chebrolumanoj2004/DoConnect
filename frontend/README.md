# DoConnect - MERN Stack Discussion Platform

DoConnect is a full-stack Question and Answer web application where users can ask questions, post answers, and interact through discussions.  
The platform includes both **User** and **Admin** roles to manage questions, answers, and discussions.

This project was developed as part of a **Full Stack Programming Capstone Project**.

---

## Features

### User Features
- User registration and login
- Ask questions on different topics
- Search questions using keywords
- Post answers to questions
- Like and comment on answers
- Participate in discussion threads
- View personal profile and activity

### Admin Features
- Admin login and authentication
- Approve questions and answers before they are visible
- Delete inappropriate questions or answers
- Manage discussion threads
- Close discussions and mark them as resolved
